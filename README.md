## Usage

Adjust live polling:

In `server.js` set `setInterval(poll, 5000);` to desired interval.

Test polling:

Send `GET` request to `/api/test`

## Requirements

- Node
- Bower
- MySQL


## Setup


Install dependencies and setup database:

```
npm install

```

For development and constant recompiling of jsx files:

```
npm run build

```

Start Server:

```
npm start

```

Visit:

```

localhost:3000

```
